==============================
DATA STRUCTURES & ALGORITHMS
==============================

1. INTRODUCTION
---------------
Data Structures and Algorithms (DSA) are fundamental concepts in computer science.
Data Structures help in organizing data efficiently, while Algorithms define step-by-step
procedures to solve problems.

Importance:
- Efficient data handling
- Optimized performance
- Core for coding interviews and development

----------------------------------

2. TIME AND SPACE COMPLEXITY
----------------------------
Used to measure performance of algorithms.

Big-O Notation:
- O(1)  : Constant Time
- O(n)  : Linear Time
- O(log n) : Logarithmic Time
- O(n^2) : Quadratic Time

Example:
Searching in array (linear search) → O(n)
Binary search → O(log n)

----------------------------------

3. ARRAYS
---------
Definition:
Collection of elements stored in contiguous memory locations.

Operations:
- Access: O(1)
- Insertion: O(n)
- Deletion: O(n)

Types:
- 1D Array
- 2D Array

----------------------------------

4. LINKED LIST
--------------
Definition:
Collection of nodes where each node contains data and pointer.

Types:
- Singly Linked List
- Doubly Linked List
- Circular Linked List

Advantages:
- Dynamic size
Disadvantages:
- No random access

----------------------------------

5. STACK
--------
Definition:
Follows LIFO (Last In First Out)

Operations:
- Push
- Pop
- Peek

Applications:
- Expression evaluation
- Undo operations

----------------------------------

6. QUEUE
--------
Definition:
Follows FIFO (First In First Out)

Types:
- Simple Queue
- Circular Queue
- Priority Queue
- Deque

Applications:
- Scheduling
- Buffering

----------------------------------

7. TREES
--------
Definition:
Hierarchical data structure.

Types:
- Binary Tree
- Binary Search Tree (BST)
- AVL Tree
- Heap

Traversals:
- Inorder
- Preorder
- Postorder

----------------------------------

8. GRAPHS
---------
Definition:
Set of vertices and edges.

Types:
- Directed
- Undirected
- Weighted

Traversal:
- BFS (Breadth First Search)
- DFS (Depth First Search)

----------------------------------

9. SORTING ALGORITHMS
---------------------
- Bubble Sort → O(n^2)
- Selection Sort → O(n^2)
- Insertion Sort → O(n^2)
- Merge Sort → O(n log n)
- Quick Sort → O(n log n)

----------------------------------

10. SEARCHING ALGORITHMS
------------------------
- Linear Search → O(n)
- Binary Search → O(log n)

----------------------------------

11. RECURSION
-------------
Definition:
Function calling itself.

Example:
Factorial:
n! = n * (n-1)!

----------------------------------

12. DYNAMIC PROGRAMMING
-----------------------
Definition:
Solving problems by storing results of subproblems.

Concepts:
- Memoization
- Tabulation

Examples:
- Fibonacci
- Knapsack Problem

----------------------------------

13. GREEDY ALGORITHMS
---------------------
Definition:
Make locally optimal choice at each step.

Examples:
- Activity Selection
- Huffman Coding

----------------------------------

14. BACKTRACKING
----------------
Definition:
Try all possibilities and backtrack if needed.

Examples:
- N-Queens
- Sudoku Solver

----------------------------------

15. HASHING
-----------
Definition:
Mapping data to fixed-size values.

Applications:
- Hash Tables
- Dictionaries
